# Дипломная работа "Поисковая система"

## Цель
* Разработать поисковый движок, способный за кратчайшее время находить введенное пользователем слово среди pdf-файлов и упорядочивать результаты по количеству вхождений на станице документа.
* Разработать клиент-серверное приложение, обслуживающее входящие запросы клиента.
